# NYC Rat Mitigation Zone Analysis

## Project Title and Description

This project examines the association between 311 rat-related complaints and initial inspections in New York City's Rat Mitigation Zones (RMZs). The analysis focuses on the **Manhattan: East Village and Chinatown** RMZ using semiannual data from 2022 to 2025.

## Research Question and Objectives

**Primary Research Question:**  
Is the number of initial inspections higher during semiannual periods with more 311 rat-related complaints in the Manhattan: East Village and Chinatown RMZ?

**Objectives:**
1. Describe and explore the RMZ datasets across key variables
2. Examine the descriptive association between complaint burden and initial inspection activity
3. Use simulation to assess how effect size, sample size, and noise influence detectability of this association

---

## Data Source and Description

Data were obtained from the **New York City Department of Health and Mental Hygiene (NYC DOHMH)** as part of the Rat Mitigation Zone initiative. RMZs are designated areas with elevated rat activity where municipal agencies concentrate inspection and mitigation efforts.

**Four datasets are used:**

| File | Description |
|------|-------------|
| `311-complaints.csv` | Semiannual 311 rat-related complaint counts by RMZ |
| `initial-inspections.csv` | Semiannual initial inspection counts by RMZ |
| `compliance-inspections.csv` | Semiannual compliance inspection counts by RMZ |
| `exterminator-visits.csv` | Semiannual exterminator visit counts by RMZ |

**Time period:** June 2022 – June 2025 (semiannual reporting periods)  
**Geographic scope:** All NYC RMZs; primary analysis restricted to Manhattan: East Village and Chinatown

**Data links:**
- https://a816-dohbesp.nyc.gov/IndicatorPublic/data-features/rat-mitigation-zones/311-complaints.csv
- https://a816-dohbesp.nyc.gov/IndicatorPublic/data-features/rat-mitigation-zones/initial-inspections.csv
- https://a816-dohbesp.nyc.gov/IndicatorPublic/data-features/rat-mitigation-zones/compliance-inspections.csv
- https://a816-dohbesp.nyc.gov/IndicatorPublic/data-features/rat-mitigation-zones/exterminator-visits.csv

## How to Run

1. Clone or download this repository.
2. Open R or RStudio and set your working directory to the repository root.
3. Install required packages if not already installed:
   ```r
   install.packages(c("readr", "dplyr", "ggplot2", "lubridate",
                      "scales", "tidyr", "purrr", "knitr"))
   ```
4. Run `scripts/01_data_exploration.R` for data loading, summary statistics, and visualizations.
5. Run `scripts/02_simulation.R` for the simulation study.

All file paths in the scripts are **relative to the repository root** and do not require modification.
